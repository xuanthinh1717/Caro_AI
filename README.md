# 🎮 Caro AI - Khung Phát Triển Game

**Framework game Gomoku (Caro) 15x15 với hệ thống phát triển AI tích hợp.**

Dùng để học tập, phát triển AI, hoặc làm dự án nhóm.

---

## 🚀 Quick Start

```bash
# 1. Setup (xem README_SETUP.md để chi tiết)
python -m venv venv
venv\Scripts\activate           # Windows
# hoặc
source venv/bin/activate        # macOS/Linux

pip install -r requirements.txt

# 2. Chạy game
python src/main.py
```
